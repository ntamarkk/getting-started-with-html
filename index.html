GETTING STARTED WITH HTML


The Basics of the Web and HTML

The World Wide Web is a collection of text documents written in a language called HTML which stands for Hypertext Markup
Language. HTML is the basis of every web page and it binds together different types of files such as plain text, HTML,
images, videos and music.

A browser for example Microsoft Internet Explorer makes requests via the internet to servers by communicating using a protocol
called HTTP. The servers respond with files that your browser displays.

HTML

HTML documents lie at the heart of the web and is made up of the follwing;

- Plain text
- Markup or tags
- Images, videos and music as examples
- Links to other pages

HTML Tags

The structure of an HTML tag looks like this < NAME>Contents< /NAME>

An entire HTML tag structure can also be referred to as an element. An HTML element refers to everything within a set of opening
and closing tags. Examples of HTML tags include;

< b> bold
< em> emphasis - makes things italic
< a> anchor - uses an attribute and is for making links. please click on this link Udacity to access the Udacity homepage.
< img> images - for including images
< br> break - for creating line breaks
< p> paragraph - another way of creating line breaks
< div> division - used to group block elements for formatting
< form> form - used to create an HTML form for completion by a user

Some HTML tags are called VOID tags because they have no content and as such, they do not need a closing tag. Examples of
void tags are < br> break and < img> images.

Inline and Block Elements

Inline elements just create a new line and examples include bold, emphasis, images, break, anchor, span and strong. on the other
hand Block elements create an invisible box (which can have height and width) around text. Examples include paragraph,
division and form.

Container Elements

Container elements only contain their elements. However, styles can be attached to them which affects how they are displayed.
Examples include span and division. They are both similar in that they can have content but a difference lies in the fact that
span is an inline element whereas division is a block element.
